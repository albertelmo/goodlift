## 알림 설정이 안 될 때 안내 문구

### 아이폰(Safari/PWA) 안내
알림 설정이 되지 않는 경우 아래를 확인해주세요.

- 설정 > Safari > 팝업 차단이 켜져 있으면 권한 요청이 막힐 수 있습니다.
- 설정 > Safari > 알림에서 이 사이트의 알림 권한이 “허용”인지 확인해주세요.
- 이미 한 번 “차단”을 선택했다면, 설정에서 “허용”으로 바꾼 후 다시 시도해주세요.
- 같은 아이폰이라도 다른 브라우저/프로필에서는 알림이 별도로 설정됩니다.
- 네트워크가 불안정하면 알림 등록이 실패할 수 있으니 다시 시도해주세요.

### 안드로이드(Chrome) 안내
알림 설정이 되지 않는 경우 아래를 확인해주세요.

- 주소창에 자물쇠(사이트 설정)에서 알림 권한이 “허용”인지 확인해주세요.
- 휴대폰 설정 > 앱 > Chrome > 알림에서 알림이 꺼져 있으면 켜주세요.
- 배터리 절약/백그라운드 제한이 강하면 알림이 늦거나 누락될 수 있습니다.
- 기기/브라우저 업데이트가 오래된 경우 알림 등록이 실패할 수 있습니다.
- 앱(브라우저) 캐시 삭제나 재설치 후 다시 시도해주세요.
