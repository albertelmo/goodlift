# 📱 회원 탭 모바일 UX 개선안

> **문제**: 모바일에서 회원 추가 폼이 항상 위에 있어 목록 확인이 불편함

---

## 🔍 현재 구조 분석

### PC 화면 (900px 이상):
```
┌─────────────────────────────────────────┐
│        [오늘] [상담] [재등록]...         │
│                                          │
│  ┌──────────┐  ┌──────────────────────┐ │
│  │          │  │     회원 목록         │ │
│  │  회원    │  ├───────┬──────┬──────┤ │

│  │  폼      │  ├───────┼──────┼──────┤ │
│  │  (200px) │  │ 홍길동 │ 강남 │ 10   │ │
```

- ❌ 실제 사용 빈도: 목록 조회 80% vs 추가 20%

## 💻 구현 방법 (개선안 1)

### HTML 구조 변경 (`public/js/main.js`):

**Before**:
```html
<div class="member-flex-wrap">
  <div id="member-add"></div>
  <div id="member-list"></div>
</div>
```

**After**:
```html
<div class="member-container">
  <!-- 모바일 전용 탭 -->
  <div class="member-mobile-tabs">
    <button class="member-tab-btn active" data-tab="list">
      📋 목록
    </button>
    <button class="member-tab-btn" data-tab="add">
      ➕ 추가
    </button>
  </div>
  
  <!-- PC/모바일 공통 컨텐츠 -->
  <div class="member-flex-wrap">
    <div id="member-add" class="member-tab-content"></div>
    <div id="member-list" class="member-tab-content active"></div>
  </div>
</div>
```

### JavaScript 추가 (`public/js/member.js` 또는 `main.js`):

```javascript
// 모바일 탭 전환 이벤트
function initMemberMobileTabs() {
    const tabButtons = document.querySelectorAll('.member-tab-btn');
    
    tabButtons.forEach(btn => {
        btn.addEventListener('click', () => {
            const targetTab = btn.dataset.tab;
            
            // 모든 탭 버튼 비활성화
            tabButtons.forEach(b => b.classList.remove('active'));
            // 클릭한 버튼 활성화
            btn.classList.add('active');
            
            // 컨텐츠 전환
            const addContent = document.getElementById('member-add');
            const listContent = document.getElementById('member-list');
            
            if (targetTab === 'add') {
                addContent.classList.add('active');
                listContent.classList.remove('active');
            } else {
                addContent.classList.remove('active');
                listContent.classList.add('active');
            }
        });
    });
}

// 탭 렌더링 후 호출
// renderTabContent()에서 Member 탭일 때 실행
if (tabId === 'Member') {
    member.renderAddForm(document.getElementById('member-add'));
    member.renderList(document.getElementById('member-list'));
    initMemberMobileTabs(); // 추가!
}
```
